Pridemalkin // Горделивый Кот
Тип карты: Creature - Cat // Существо - Кошка
Вызов:
Сила/Защита: 2 / 1
Редкость: - Обычная
Издание: Core Set 2021 // Базовый выпуск 2021
When Pridemalkin enters the battlefield, put a +1/+1 counter on target creature you control.

Each creature you control with a +1/+1 counter on it has trample. (They can deal excess combat damage to the player or planeswalker they're attacking.)
Когда Горделивый Кот выходит на поле битвы, положите один жетон +1/+1 на целевое существо под вашим контролем.

Каждое существо под вашим контролем, на котором есть жетон +1/+1, имеет Пробивной удар. (Оно может нанести избыточные боевые повреждения игроку или planeswalker'у, которого атакует.)
